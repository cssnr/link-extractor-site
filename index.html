Modern Chrome and Firefox Addon to easily extract, parse, or open all links/domains from a site or text with optional filters. Feature packed with automatic dark/light mode, copy to clipboard, keyboard shortcuts, custom options, and much more...

  • Extract Links and/or Domains from Any Site
  • Parse Links from text/clipboard or open in tabs
  • Quick Filter Links with a regular expression
  • Quick Filter Links with saved regular expressions
  • Right-Click Context Menu which can be disabled in options
  • Keyboard Shortcuts to activate addon and copy links/domains
  • Automatic Dark/Light Mode based on browser setting

Install

Google Chrome Web Store
Mozilla Firefox Add-ons

All Chromium Based Browsers can install the extension from the Chrome Web Store.
