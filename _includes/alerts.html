Known Issues:
  • Saving Options on the Options Page does not work prior to 0.6.1
    Work Around: Set your options in the Popup.
  • Opening Links from the Parse Links Option in the Popup does not work prior to 0.6.1
    Work Around: Use Open Lines or Parse URLs then Open from Links Results.

Chrome Web Store Publishing can take several days (and sometimes weeks) to complete. These issues have already been fixed; however, are still awaiting approval from Google.
