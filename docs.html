Configure

You can access Link Extractor through the Icon on the Extension Toolbar or Right Click Context Menu.

You can pin the Addon by locating the Link Extractor icon on the Extension Toolbar, then;

  • Firefox, click the Settings Wheel and Pin to Toolbar.
  • Chrome, click the Pin icon.

Host Permissions

In order to extract links from multiple tabs, the web extension needs host permissions granted. This can be granted from popup window or on the Options page. Firefox also requires Host Permissions for Omnibox usage.

Note: This extension does NOT inject a content-script into every site and only injects a script when you extract links. Therefore, granting host permissions does not reduce overall performance in any way.

Tip: You can select multiple tabs by holding the Ctrl key and selecting additional tabs.

Usage

Activate with any one of these 4 ways:

  • Toolbar Icon Popup
  • Keyboard Shortcuts
  • Right Click Context Menu
  • Omnibox Keyword: link

Tip: You may also select text with your mouse, and right click to extract links from the selection.

Omnibox

Keyword: link {% comment %}{% endcomment %}

Omnibox lets you extract links by typing link into the address bar followed by a space.
Additionally you can start typing to show saved filters matching your entry or enter an on-the-fly filter.
Chrome does not allow an empty omnibox entry. To extract all links, simply enter a second space after activating the Omnibox.
Firefox does not allow using the Omnibox unless you have granted host permissions to the web extension.

Options

Keyboard Shortcuts

These can be configured on your browser's extension management page and varies by browser.
Firefox: https://support.mozilla.org/en-US/kb/manage-extension-shortcuts-firefox
Chrome: Google does not provide documentation on extension shortcuts. However, you can manage these by entering the following into your address bar: chrome://extensions/shortcuts {% comment %}{% endcomment %}

Regex Flags

This is the default flags used when filtering links via quick filter or a saved filter. For more information read this document.

Remove Duplicate Links

Remove duplicate links from the results page. Disable this to see all links on the page for auditing and extra fields.

Use Default Link Filtering

Filter out any links that do not contain ://. Turn this off this to include links such as magnet: and mailto: etc.

Save Links Page Options

Save Links Page settings, such as number of links shown, extra fields displayed, and column sorting.

Enable Right Click Menu

Enable or Disable Right Click Context Menu.

Show Release Notes on Update

Open the Release Notes after a new update has been installed.

Filters

Add as many saved filters as you need. They will show up on the Popup Filter Dropdown, the Right Click Context Menu and Omnibox suggestions.
Edit an existing filter by clicking on it.
Reorder filters by clicking on the grip and dragging.
