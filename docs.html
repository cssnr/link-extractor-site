Configure

You can access Link Extractor through the Icon on the Extension Toolbar or Right Click Context Menu.

You can pin the Addon by locating the Link Extractor icon on the Extension Toolbar, then;

  • Firefox, click the Settings Wheel and Pin to Toolbar.
  • Chrome, click the Pin icon.

Usage

Activate 1 of 3 ways:

  • Toolbar Icon Popup
  • Right Click Context Menu
  • Keyboard Shortcuts for Extract/Popup

Extract all links, just domains, use a filter, or a saved filter.

You may also select text with your mouse, and right click to extract links from the selection.

Options

Keyboard Shortcuts

These can be configured on your browser's extension management page and varies by browser.

Regex Flags

This is the default flags used when filtering links via quick filter or a saved filter. For more information read this document.

Filters

Add as many saved filters as you need. They will show up on the Popup Filter Dropdown, and on the Right Click Context Menu.

Enable Right Click Menu

Enable or Disable Right Click Context Menu.

Use Default Link Filtering

Filter out any links that do not contain ://. Turn this off this to include links such as magnet: and mailto: etc.

Show Release Notes on Update

Open the Release Notes after a new update has been installed.
