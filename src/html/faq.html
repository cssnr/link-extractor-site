No links are extracted?

  1. If you just installed the extension, you may need to close and re-open the tab to extract links.
  2. Try re-starting your browser. Make sure to close all open windows then re-open the browser.
  3. If you cant extract links from any tabs try re-installing the extension. Make sure to export any saved filters first.
  4. If you are still unable to extract links, please open an issue on GitHub and we will do our best to resolve the issue.
If you don't have a GitHub account, you can submit feedback here .
Make sure to include a link to the site you are having issues with.

Extract links from multiple tabs at the same time?

  1. Ensure you have granted Host Permission to the extension, from the Popup or Options Page. Screen Shot.
  2. While pressing and holding the Ctrl key, select all the tabs you wish to extract from.
  3. With the tabs highlighted, extract links as normal using either the Popup, Shortcuts, or Omnibox.

The link results page should have results from all the tabs you selected.

Allow access to incognito or private browsing windows?

  1. Navigate to about:addons or press Ctrl+Shift+A
  2. Scroll until you find Link Extractor and click on the tile.
  3. Scroll until you find Run in Private Windows and toggle the Allow radio.

The extension should now load in new Private Windows.

  1. Navigate to: chrome://extensions
  2. Scroll until you find Link Extractor and click Details.
  3. Scroll until you find Allow in Incognito and enable the toggle.

The extension should now load in new Incognito Windows.

  1. Navigate to: edge://extensions
  2. Scroll until you find Link Extractor and click Details.
  3. Check the box labeled Allow in InPrivate.

The extension should now load in new IniPrivate Windows.

Change order of extracted or opened links?

Links are extracted in the order they are found on the page; however, they are sorted in alphabetical order by default.

To re-order the links, click on the Links heading to cycle through these sort orders:

  • Order they are found on the page.
  • Alphabetical order.
  • Reverse alphabetical order.

Tip: To remember sort order across extractions make sure the Save Links Page Options option is enabled.

Unable to extract links from PDF documents?

Extracting links from PDF documents is currently experimental.

  1. PDF Extraction utilizes the pdf.js library which requires Firefox 121 or newer. Chrome 119 or newer. Edge 119 or newer.
  2. Ensure you have granted Host Permission to the extension, from the Popup or Options Page. Screen Shot.
  3. Try re-starting your browser. Make sure to close all open windows then re-open the browser.
  4. If you have tried all the above steps, please open an issue on GitHub and we will do our best to resolve the issue.

PDF Files

  1. To extract links from PDF files, you must also grant File Access.

Allow access to file:/ URLs?

File extraction is currently experimental and only works on PDF files.

Firefox does not allow access to file:/ URLs.
  1. Navigate to: chrome://extensions
  2. Scroll until you find Link Extractor and click Details.
  3. Scroll until you find Allow access to file URLs and enable the toggle.
  1. Navigate to: edge://extensions
  2. Scroll until you find Link Extractor and click Details.
  3. Check the box labeled Allow access to file URLs.

You may need to restart the browser, reload the tab, or wait a few minutes for this to take effect.
